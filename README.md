# Sanctuario de Santa Rosa de Lima Memorial Park Management System

A comprehensive web-based management system for the Sanctuario de Santa Rosa de Lima Memorial Park, featuring role-based access control, grave management, amortization tracking, and client portals.

## Features

### 🏠 Public Homepage
- Landing page with services showcase
- About section with mission and vision
- Contact information and inquiry form
- Client registration and login

### 👨‍💼 Admin Dashboard
- Complete system management
- Grave management with interactive map
- Amortization tracking
- Inquiry management
- Maintenance scheduling
- User management and settings

### 👷 Staff Dashboard
- Limited operational access
- Grave management
- Basic reporting
- Maintenance tasks

### 👤 Client Dashboard
- Personal grave information
- Amortization status
- Inquiry submission and tracking
- Profile management

## Technology Stack
- **Backend**: PHP 8.0+
- **Database**: MySQL 8.0+
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **UI Framework**: Bootstrap 5
- **Icons**: Font Awesome
- **Charts**: Chart.js
- **Deployment**: XAMPP compatible

## Installation

1. Clone this repository to your XAMPP `htdocs` folder
2. Create a MySQL database named `sanctuario_db`
3. Import the database schema from `database/schema.sql`
4. Configure database connection in `config/database.php`
5. Access the system at `http://localhost/sanctuario`

## User Roles

### Admin
- Full system access
- User management
- System configuration
- All operational features

### Staff
- Limited access to operational tasks
- Grave management
- Basic reporting

### Client
- Personal account features
- View own graves and payments
- Submit inquiries
