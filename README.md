# Sanctuario de Santa Rosa de Lima Memorial Park Website

A modern, elegant website for Sanctuario de Santa Rosa de Lima Memorial Park featuring a beautiful design with 2025 styling standards.

## 🌟 Features

### Core Features
- **Modern Design**: Clean, elegant interface with peaceful color palette
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Subtle animations and transitions for enhanced UX
- **Memorial Lot Showcase**: Detailed information about all available lots
- **Contact Integration**: Multiple contact methods and inquiry forms
- **Authentication System**: Login and signup functionality

### Memorial Lot Types
1. **Lawn Lots** (2.38 sq m) - Underground interment, flat tombstone
2. **Garden Lots** (16 sq m) - Underground interment, enhanced space
3. **Mini Mausoleum Lot** (21 sq m) - Underground structure, uniform design
4. **Mausoleum Lot** (30 sq m) - Premium option with flexible interment
5. **Rental Apartment Niche** (0.664 sq m) - Above ground, one-time payment

### Interactive Elements
- **Smooth Scrolling Navigation**: Seamless page navigation
- **Modal Windows**: Login/signup forms in elegant overlays
- **Contact Form**: Pre-filled with lot type selection
- **Mobile Menu**: Hamburger menu for mobile devices
- **Notification System**: User feedback for form submissions
- **Parallax Effects**: Subtle background movement on scroll

## 🚀 Getting Started

### Simple Setup
1. Download all files to a folder
2. Open `index.html` in any modern web browser
3. The website is ready to use!
