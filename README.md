# Sanctuario de Santa Rosa de Lima Memorial Park Website

A beautiful and elegant homepage for Sanctuario de Santa Rosa de Lima Memorial Park, featuring modern design, working authentication system, and responsive layout.

## Features

### 🎨 Design & User Experience
- **Elegant Design**: Sophisticated typography using Playfair Display and Crimson Text fonts
- **Comfort & Elegance Theme**: Soothing color palette with purple gradients and peaceful aesthetics
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile devices
- **Smooth Animations**: Fade-in animations, hover effects, and smooth scrolling navigation
- **Professional Layout**: Well-structured sections including hero, about, services, memorial, and contact

### 🔐 Authentication System
- **Login Modal**: Clean login form with email/password authentication
- **Registration Modal**: Comprehensive signup form with form validation
- **Form Validation**: Real-time validation for email, password strength, phone numbers
- **Remember Me**: Option to remember user sessions using localStorage
- **Switch Between Forms**: Easy switching between login and signup modals
- **User Session Management**: Basic user state management with logout functionality

### 📱 Interactive Features
- **Working Contact Form**: Functional contact form with validation and submission feedback
- **Modal System**: Elegant modal windows for authentication
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Scroll Effects**: Dynamic navbar styling based on scroll position
- **Notifications**: Toast notifications for user feedback
- **Keyboard Navigation**: Support for ESC key to close modals

### 🏛️ Cemetery-Specific Content
- **Memorial Services**: Information about burial, cremation, and memorial services
- **Sacred Heritage**: Content highlighting the connection to Santa Rosa de Lima
- **Compassionate Care**: Emphasis on comfort and support during difficult times
- **Garden Memorials**: Beautiful garden settings for eternal rest
- **Contact Information**: Complete contact details and hours

## File Structure

```
├── index.html          # Main HTML file with complete page structure
├── styles.css          # Comprehensive CSS with responsive design
├── script.js           # JavaScript for interactivity and authentication
└── README.md           # Project documentation
```

## Technologies Used

- **HTML5**: Semantic markup with accessibility considerations
- **CSS3**: Modern CSS with Flexbox, Grid, animations, and responsive design
- **Vanilla JavaScript**: No external frameworks, pure JavaScript functionality
- **Font Awesome**: Icons for enhanced visual appeal
- **Google Fonts**: Playfair Display and Crimson Text for elegant typography

## Getting Started

1. **Clone or download** the project files
2. **Open `index.html`** in a web browser
3. **No server required** - runs directly in the browser

## Features in Detail

### Authentication System
- **Login**: Users can log in with email and password
- **Registration**: New users can create accounts with full validation
- **Validation**: Real-time form validation including:
  - Email format validation
  - Password strength requirements (8+ characters, uppercase, lowercase)
  - Password confirmation matching
  - Phone number format validation
  - Required field validation

### Responsive Design
- **Desktop**: Full-width layout with sidebar navigation
- **Tablet**: Adjusted layouts and spacing
- **Mobile**: Hamburger menu and stacked layouts

### Interactive Elements
- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Hover Effects**: Interactive buttons and cards with hover animations
- **Loading States**: Forms show loading states during submission
- **Error Handling**: Comprehensive error messaging and validation

## Customization

### Colors
The color scheme can be modified in `styles.css`:
- Primary purple: `#6b73ff`
- Secondary purple: `#9b59b6`
- Dark text: `#2c3e50`
- Light background: `#f8f9fa`

### Content
All content can be customized in `index.html`:
- Cemetery name and tagline
- Service descriptions
- Contact information
- Footer links

### Fonts
Fonts are loaded from Google Fonts and can be changed in the HTML head section.
